# Linux 기초 개념 및 명령어

## Linux 운영체제 개념

### Linux란?
#### 기본 특징:
- **오픈소스**: 소스 코드가 공개된 자유 소프트웨어
- **다중 사용자**: 여러 사용자가 동시에 시스템 사용 가능
- **다중 작업**: 여러 프로그램을 동시에 실행 가능
- **이식성**: 다양한 하드웨어 플랫폼에서 실행 가능
- **안정성**: 서버 환경에서 장기간 안정적 운영

#### 주요 배포판:
- **Ubuntu**: 데스크톱 및 서버용으로 인기
- **CentOS/RHEL**: 기업 환경에서 널리 사용
- **Debian**: 안정성을 중시하는 서버용
- **Alpine**: 컨테이너용 경량 배포판

### 커널과 쉘
#### 커널 (Kernel):
- **역할**: 하드웨어와 소프트웨어 간 인터페이스
- **기능**: 메모리 관리, 프로세스 스케줄링, 파일 시스템 관리
- **종류**: 모놀리식 커널 (Linux), 마이크로 커널

#### 쉘 (Shell):
- **정의**: 사용자와 커널 간의 인터페이스
- **종류**: bash, zsh, fish, sh
- **기능**: 명령어 해석, 스크립트 실행, 환경 변수 관리

## Linux 파일 시스템 구조

### 디렉토리 계층 구조
#### 루트 디렉토리 (`/`):
- **최상위 디렉토리**: 모든 파일과 디렉토리의 시작점
- **절대 경로**: `/`로 시작하는 경로 (예: `/home/user`)
- **상대 경로**: 현재 위치 기준 경로 (예: `../documents`)

#### 주요 시스템 디렉토리:
- **`/bin`**: 기본 실행 파일 (ls, cp, mv 등)
- **`/sbin`**: 시스템 관리자용 실행 파일 (mount, ifconfig 등)
- **`/usr`**: 사용자 프로그램과 라이브러리
- **`/usr/bin`**: 사용자 명령어
- **`/usr/local`**: 로컬에서 설치한 프로그램
- **`/lib`**: 공유 라이브러리 파일
- **`/etc`**: 시스템 설정 파일
- **`/var`**: 가변 데이터 (로그, 캐시, 스풀)
- **`/tmp`**: 임시 파일 저장소
- **`/proc`**: 프로세스 정보 (가상 파일 시스템)
- **`/sys`**: 시스템 정보 (가상 파일 시스템)

#### 사용자 관련 디렉토리:
- **`/home`**: 일반 사용자 홈 디렉토리
- **`/root`**: root 사용자 홈 디렉토리
- **`/boot`**: 부팅 관련 파일
- **`/dev`**: 장치 파일 (하드웨어 인터페이스)
- **`/mnt`**: 임시 마운트 포인트
- **`/media`**: 이동식 미디어 마운트 포인트

### 파일 타입과 권한
#### 파일 타입:
- **일반 파일**: `-` (텍스트, 바이너리 파일)
- **디렉토리**: `d` (폴더)
- **심볼릭 링크**: `l` (바로가기)
- **장치 파일**: `c` (문자), `b` (블록)
- **파이프**: `p` (FIFO)
- **소켓**: `s` (네트워크 통신)

#### 권한 시스템:
- **소유자 (Owner)**: 파일을 생성한 사용자
- **그룹 (Group)**: 파일이 속한 그룹
- **기타 (Others)**: 나머지 모든 사용자
- **권한 종류**: 읽기(r), 쓰기(w), 실행(x)
- **숫자 표현**: 4(읽기) + 2(쓰기) + 1(실행) = 7(모든 권한)

## 기본 파일 및 디렉토리 명령어

### 디렉토리 탐색
#### 기본 탐색 명령어:
- **`pwd`**: 현재 작업 디렉토리 출력 (Print Working Directory)
- **`cd [경로]`**: 디렉토리 변경 (Change Directory)
  - `cd ~` 또는 `cd`: 홈 디렉토리로 이동
  - `cd ..`: 상위 디렉토리로 이동
  - `cd -`: 이전 디렉토리로 이동
- **`ls [옵션] [경로]`**: 디렉토리 내용 나열 (List)
  - `ls -l`: 상세 정보 표시 (권한, 소유자, 크기, 날짜)
  - `ls -a`: 숨김 파일 포함 표시
  - `ls -la`: 상세 정보 + 숨김 파일
  - `ls -h`: 파일 크기를 읽기 쉬운 형태로 표시

### 파일 및 디렉토리 조작
#### 생성 명령어:
- **`mkdir [디렉토리명]`**: 디렉토리 생성 (Make Directory)
  - `mkdir -p path/to/dir`: 중간 디렉토리까지 함께 생성
- **`touch [파일명]`**: 빈 파일 생성 또는 타임스탬프 갱신
- **`rmdir [디렉토리명]`**: 빈 디렉토리 삭제

#### 복사 및 이동:
- **`cp [원본] [대상]`**: 파일 복사 (Copy)
  - `cp -r`: 디렉토리 재귀적 복사
  - `cp -i`: 덮어쓰기 전 확인
  - `cp -p`: 권한과 타임스탬프 유지
- **`mv [원본] [대상]`**: 파일/디렉토리 이동 또는 이름 변경 (Move)
- **`ln -s [원본] [링크명]`**: 심볼릭 링크 생성

#### 삭제 명령어:
- **`rm [파일명]`**: 파일 삭제 (Remove)
  - `rm -r`: 디렉토리 재귀적 삭제
  - `rm -f`: 강제 삭제 (확인 없이)
  - `rm -i`: 삭제 전 확인
  - `rm -rf`: 강제로 디렉토리 전체 삭제 (주의!)

### 파일 내용 보기
#### 텍스트 파일 보기:
- **`cat [파일명]`**: 파일 전체 내용 출력
- **`less [파일명]`**: 페이지 단위로 파일 내용 보기 (q로 종료)
- **`more [파일명]`**: 페이지 단위로 파일 내용 보기 (구식)
- **`head [파일명]`**: 파일 앞부분 10줄 출력
  - `head -n 20`: 앞부분 20줄 출력
- **`tail [파일명]`**: 파일 뒷부분 10줄 출력
  - `tail -n 20`: 뒷부분 20줄 출력
  - `tail -f`: 실시간으로 파일 끝 부분 모니터링

#### 파일 검색:
- **`find [경로] [조건]`**: 파일 및 디렉토리 검색
  - `find /home -name "*.txt"`: txt 파일 검색
  - `find . -type f -size +100M`: 100MB 이상 파일 검색
- **`locate [파일명]`**: 파일명 데이터베이스에서 빠른 검색
- **`which [명령어]`**: 명령어의 실행 파일 위치 찾기
- **`whereis [명령어]`**: 명령어, 소스, 매뉴얼 위치 찾기

## 텍스트 처리 명령어

### 텍스트 검색 및 필터링
#### grep 명령어:
- **`grep [패턴] [파일]`**: 텍스트 패턴 검색
  - `grep -i`: 대소문자 구분 안함
  - `grep -r`: 디렉토리 재귀 검색
  - `grep -n`: 줄 번호 표시
  - `grep -v`: 패턴과 일치하지 않는 줄 출력

#### 텍스트 정렬 및 중복 제거:
- **`sort [파일]`**: 텍스트 정렬
  - `sort -n`: 숫자 정렬
  - `sort -r`: 역순 정렬
- **`uniq [파일]`**: 중복된 줄 제거 (정렬된 파일에서)
- **`wc [파일]`**: 단어, 줄, 문자 수 계산
  - `wc -l`: 줄 수만 출력
  - `wc -w`: 단어 수만 출력

#### 텍스트 편집:
- **`sed`**: 스트림 에디터 (텍스트 치환, 삭제)
  - `sed 's/old/new/g'`: 모든 'old'를 'new'로 치환
- **`awk`**: 패턴 스캐닝 및 처리 언어
  - `awk '{print $1}'`: 첫 번째 필드만 출력
- **`cut`**: 텍스트 컬럼 추출
  - `cut -d':' -f1`: ':' 구분자로 첫 번째 필드 추출

## 권한 및 소유권 관리

### 파일 권한 변경
#### chmod 명령어:
- **`chmod [권한] [파일]`**: 파일 권한 변경
  - `chmod 755 file`: rwxr-xr-x (소유자: 모든 권한, 그룹/기타: 읽기+실행)
  - `chmod +x file`: 실행 권한 추가
  - `chmod -w file`: 쓰기 권한 제거
  - `chmod u+w,g-r file`: 소유자에게 쓰기 추가, 그룹에서 읽기 제거

#### 권한 표기법:
- **숫자 표기**: 4(r) + 2(w) + 1(x)
  - 755 = rwxr-xr-x
  - 644 = rw-r--r--
  - 600 = rw-------
- **문자 표기**: u(user), g(group), o(others), a(all)
  - +: 권한 추가
  - -: 권한 제거
  - =: 권한 설정

### 소유권 변경
#### chown 명령어:
- **`chown [소유자]:[그룹] [파일]`**: 소유자 및 그룹 변경
  - `chown user:group file`: 소유자와 그룹 모두 변경
  - `chown user file`: 소유자만 변경
  - `chown :group file`: 그룹만 변경
  - `chown -R user:group dir/`: 디렉토리 재귀적 변경

#### chgrp 명령어:
- **`chgrp [그룹] [파일]`**: 그룹 소유권만 변경

## 프로세스 관리

### 프로세스 확인
#### 프로세스 목록:
- **`ps`**: 현재 실행 중인 프로세스 표시
  - `ps aux`: 모든 프로세스 상세 정보
  - `ps -ef`: 모든 프로세스 전체 형식
  - `ps -u user`: 특정 사용자의 프로세스
- **`top`**: 실시간 프로세스 모니터링
  - CPU, 메모리 사용률 순으로 정렬
  - q: 종료, k: 프로세스 종료
- **`htop`**: top의 개선된 버전 (설치 필요)

#### 프로세스 검색:
- **`pgrep [프로세스명]`**: 프로세스 ID 검색
- **`pidof [프로세스명]`**: 프로세스 ID 찾기
- **`pstree`**: 프로세스 트리 구조 표시

### 프로세스 제어
#### 프로세스 종료:
- **`kill [PID]`**: 프로세스 종료 (SIGTERM)
- **`kill -9 [PID]`**: 강제 종료 (SIGKILL)
- **`killall [프로세스명]`**: 이름으로 프로세스 종료
- **`pkill [패턴]`**: 패턴 매칭으로 프로세스 종료

#### 백그라운드 실행:
- **`command &`**: 명령어를 백그라운드에서 실행
- **`nohup command &`**: 로그아웃 후에도 계속 실행
- **`jobs`**: 백그라운드 작업 목록 확인
- **`fg [작업번호]`**: 백그라운드 작업을 포그라운드로
- **`bg [작업번호]`**: 정지된 작업을 백그라운드로

## 시스템 정보 및 모니터링

### 시스템 정보 확인
#### 기본 시스템 정보:
- **`uname -a`**: 시스템 정보 (커널, 호스트명, 아키텍처)
- **`hostname`**: 호스트명 확인
- **`whoami`**: 현재 사용자명 확인
- **`id`**: 사용자 ID 및 그룹 정보
- **`date`**: 현재 날짜 및 시간
- **`uptime`**: 시스템 가동 시간 및 부하

#### 하드웨어 정보:
- **`lscpu`**: CPU 정보 확인
- **`lsmem`**: 메모리 정보 확인
- **`lsblk`**: 블록 장치 (디스크) 정보
- **`lsusb`**: USB 장치 정보
- **`lspci`**: PCI 장치 정보

### 디스크 및 메모리 사용량
#### 디스크 사용량:
- **`df -h`**: 파일 시스템 디스크 사용량 (사람이 읽기 쉬운 형태)
- **`du -h [경로]`**: 디렉토리별 디스크 사용량
  - `du -sh`: 총 사용량만 표시
  - `du -ah`: 모든 파일 크기 표시

#### 메모리 사용량:
- **`free -h`**: 메모리 사용량 (사람이 읽기 쉬운 형태)
- **`vmstat`**: 가상 메모리 통계
- **`iostat`**: I/O 통계 (sysstat 패키지 필요)

## 네트워크 명령어

### 네트워크 설정 확인
#### 네트워크 인터페이스:
- **`ifconfig`**: 네트워크 인터페이스 설정 확인 (구식)
- **`ip addr show`**: 네트워크 인터페이스 정보 (최신)
- **`ip route show`**: 라우팅 테이블 확인
- **`netstat -tuln`**: 네트워크 연결 상태 확인
- **`ss -tuln`**: netstat의 최신 대체 명령어

#### 네트워크 연결 테스트:
- **`ping [호스트]`**: 네트워크 연결 테스트
  - `ping -c 4`: 4번만 ping 전송
- **`wget [URL]`**: 파일 다운로드
- **`curl [URL]`**: HTTP 요청 전송
- **`nslookup [도메인]`**: DNS 조회
- **`dig [도메인]`**: DNS 상세 조회

## 아카이브 및 압축

### 압축 파일 관리
#### tar 명령어:
- **`tar -cvf archive.tar files/`**: tar 아카이브 생성
- **`tar -xvf archive.tar`**: tar 아카이브 추출
- **`tar -czvf archive.tar.gz files/`**: gzip 압축과 함께 아카이브
- **`tar -xzvf archive.tar.gz`**: gzip 압축 아카이브 추출
- **`tar -tf archive.tar`**: 아카이브 내용 확인

#### 기타 압축 도구:
- **`gzip [파일]`**: gzip 압축
- **`gunzip [파일.gz]`**: gzip 압축 해제
- **`zip -r archive.zip files/`**: zip 압축
- **`unzip archive.zip`**: zip 압축 해제

## 패키지 관리

### 패키지 관리자 (Ubuntu/Debian)
#### apt 명령어:
- **`sudo apt update`**: 패키지 목록 업데이트
- **`sudo apt upgrade`**: 설치된 패키지 업그레이드
- **`sudo apt install [패키지명]`**: 패키지 설치
- **`sudo apt remove [패키지명]`**: 패키지 제거
- **`apt search [키워드]`**: 패키지 검색
- **`apt show [패키지명]`**: 패키지 정보 확인

### 패키지 관리자 (CentOS/RHEL)
#### yum/dnf 명령어:
- **`sudo yum update`**: 패키지 업데이트
- **`sudo yum install [패키지명]`**: 패키지 설치
- **`sudo yum remove [패키지명]`**: 패키지 제거
- **`yum search [키워드]`**: 패키지 검색

## 환경 변수 및 PATH

### 환경 변수 관리
#### 환경 변수 확인:
- **`env`**: 모든 환경 변수 출력
- **`echo $PATH`**: PATH 환경 변수 확인
- **`echo $HOME`**: 홈 디렉토리 경로 확인
- **`printenv [변수명]`**: 특정 환경 변수 값 확인

#### 환경 변수 설정:
- **`export VAR=value`**: 환경 변수 설정 (현재 세션)
- **`~/.bashrc`**: bash 쉘 설정 파일 (사용자별)
- **`~/.profile`**: 로그인 시 실행되는 설정 파일
- **`/etc/environment`**: 시스템 전체 환경 변수

## 도움말 및 매뉴얼

### 명령어 도움말
#### 매뉴얼 페이지:
- **`man [명령어]`**: 명령어 매뉴얼 페이지 보기
- **`info [명령어]`**: 상세한 정보 문서
- **`[명령어] --help`**: 간단한 도움말 출력
- **`whatis [명령어]`**: 명령어 간단 설명
- **`apropos [키워드]`**: 키워드 관련 명령어 검색
