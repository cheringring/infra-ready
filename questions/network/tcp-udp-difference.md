# TCP와 UDP 비교

## TCP (Transmission Control Protocol)
### 특징
- **연결 지향적**: 3-way handshake로 연결 설정
- **신뢰성 보장**: 데이터 순서 보장, 오류 검출 및 재전송
- **흐름 제어**: 수신자의 처리 속도에 맞춰 전송 속도 조절
- **혼잡 제어**: 네트워크 상황에 따라 전송량 조절

### 사용 예시
- **웹 브라우징** (HTTP/HTTPS)
- **이메일** (SMTP, POP3, IMAP)
- **파일 전송** (FTP)
- **원격 접속** (SSH, Telnet)

## UDP (User Datagram Protocol)
### 특징
- **비연결 지향적**: 연결 설정 없이 바로 전송
- **빠른 전송**: 오버헤드가 적어 속도가 빠름
- **신뢰성 미보장**: 데이터 손실 가능성 있음
- **단순한 구조**: 헤더가 간단함

### 사용 예시
- **실시간 스트리밍** (동영상, 음성)
- **DNS 조회**
- **온라인 게임**
- **DHCP**

## 면접 포인트
1. **언제 TCP를 선택하나요?**
   - 데이터 정확성이 중요한 경우
   - 순서가 중요한 경우
   - 예: 금융 거래, 파일 다운로드

2. **언제 UDP를 선택하나요?**
   - 속도가 중요한 경우
   - 실시간성이 중요한 경우
   - 예: 라이브 스트리밍, 온라인 게임
